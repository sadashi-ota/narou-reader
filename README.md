# narou-reader

**本アプリケーションはまだ開発中です。**

このアプリケーションは「[小説家になろう](http://syosetu.com/)」にて公開されている小説のリーダーアプリです。  
「なろうデベロッパー」で公開されている「[なろう小説API](https://dev.syosetu.com/man/api/)」を利用しています。
